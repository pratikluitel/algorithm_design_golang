## Excercises from Steven S. Skiena's [The Algorithm Design Manual](https://www.amazon.com/Algorithm-Design-Manual-Steven-Skiena/dp/1849967202)

The programming is done in Go, and the code is arranged on the basis of chapters, leetcode, hackerrank problems and programming challenges

## Folder structure
```bash
root
|-- chapter1
|-- chapter2
......
|-- chapterN
|-- leetcode
|    |--ch1_problemname1.go
|    |--ch1_problemname2.go
|    |--ch2_problemname1.go
|    .......
|    |--chN_problemnameN.go
|-- hackerrank
|    |--ch1_problemname1.go
|    |--ch1_problemname2.go
|    |--ch2_problemname1.go
|    .......
|    |--chN_problemnameN.go
|-- programming_challenges
     |--ch1_problemname1.go
     |--ch1_problemname2.go
     |--ch2_problemname1.go
     .......
     |--chN_problemnameN.go
```
## To run
In the root directory:

```bash
go run .
```
